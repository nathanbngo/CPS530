Problem 3: Hosting and Viewing My Project Online

In this part of the lab, I set up a website to host my project files using Netlify, a free hosting platform. I uploaded my files from the lab10 folder, which included an index.html file and other resources like images and CSS styles. After deploying the project, I received a live URL that anyone can use to view my work.

The purpose of this task was to learn how to make a website publicly accessible and ensure everything looks and works as intended. This included testing that all links, images, and styles display correctly when accessed through the browser. Hosting the project online allows others to see it from anywhere, making it easy to share and showcase.
